# Макросы
Название макроса всегда заканчивается `!`.
```rust
print!(str)          ; выводит текст на стандартное устройство вывода
println!(str)        ; выводит строку (с переводом картетки) на стандартное устройство вывода
eprint!(str)         ; выводит текст на стандартное устройство ошибок
eprintln!(str)       ; выводит строку (с переводом картетки) на стандартное устройство ошибок

assert!(cond)        ; проверяет истинность условия cond и прерывает исполнение кода при его нарушении
debug_assert!(cond)  ; то же, но выполняется только при сборке отладочной версии программы

```
