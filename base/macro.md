# Макросы
Название макроса всегда заканчивается `!`.
```rust
print!(str)          ; выводин текст на стандартное устройство вывода
println!(str)        ; выводин строку (с переводом картетки) на стандартное устройство вывода
eprint!(str)         ; выводин текст на стандартное устройство ошибок
eprintln!(str)       ; выводин строку (с переводом картетки) на стандартное устройство ошибок

assert!(cond)        ; проверяет истинность условия cond и прерывает исполнение кода при его нарушении
debug_assert!(cond)  ; то же, но выполняется только при сборке отладочной версии программы

```
